## Goal

- All IP addresses have been preconfigured for you as specified in the topology picture.
- Configure IBGP within AS 1.
- Configure EBGP between router R3 and R5.
- Configure EBGP between router R5 and R6.
- Configure EBGP between router R6 and R7.
- Configure EBGP between router R4 and R7.
- Advertise all networks from AS1, AS2, AS3 and AS4 in BGP.
- Ensure AS1 can reach any network in AS2, AS3 or AS4.
- You are only allowed to use the local preference attribute for the upcoming tasks.
- Configure router R3 so all networks learned through EBGP have a local preference of 500.
- Configure router R4 so network 6.6.6.0/24 has a local preference of 600.
- Configure router R4 so all networks from AS 4 have a local preference of 700.
- Configure router R4 so network 66.66.66.0/24 has a local preference of 800, you are not allowed to use an access-list.
